投骰方法

DND 默认使用 D20 进行投骰

优势 / 劣势

优势：Advantage， 投 2 次取最高值

劣势：Disadvantage，投 2 次取最低值

优势 与 劣势 可互相抵消，但不会与自身叠加累计

力竭

力竭：Exhaustion，属性检定、攻击检定、豁免检定 都会减去 力竭层数 * 2

大成功 / 大失败

D20 = 20 为 大成功，视为自动成功 / 重击

D20 = 1 为 大失败，视为自动失败

技能检定

严格来说，大成功/大失败，尤其是技能检定原本并不存在这个概念，并不保证想做的完成，而是事情进行得相当顺利/糟糕

比如，哪怕是大成功，也不能徒手把月亮抓住，但也许顺利地想清楚了，这种事情并不可能

或者，虽然没能让国王答应离谱的请求，但至少没有直接下令为冒犯杀头

角色数据

熟练加值 Proficiency Bonus = 角色等级 / 4 + 1

生命

生命 Health Point = 角色等级 * 体质调整值 + 职业生命骰 * 职业等级

生命骰 Health Dice = (职业生命骰 + 体质调整值) * 消耗个数

死亡豁免 Death Saving Throw = D20

D20********
= 20恢复 1 HP恢复战斗状态
>= 10记 1 次成功3 次时进入 伤势稳定 状态
< 10记 1 次失败3 次时死亡
= 1记 2 次失败3 次时死亡

稳定伤势 Stabilize = 感知检定（医药） > 10

治疗药水 Potion of Healing Tier 1-4 = 2d4 + 2 / 4d4 +4 / 8d4 + 8 / 10d4 + 20

属性

属性检定 Ability Check = D20 + （熟练加值） + 属性调整值

豁免检定 Saving Throw = D20 + （熟练加值） + 属性调整值

扮演骰点

技能

技能检定 Skill = D20 + （熟练加值） + 属性调整值

（实际上，技能检定 属于 属性检定）

负重

生物体型承载拖拽/抬起/推动
微型力量 x 7.5 lb力量 x 15 lb
小型/中型力量 x 15 lb力量 x 30 lb
大型力量 x 30 lb力量 x 60 lb
巨型力量 x 60 lb力量 x 120 lb
超巨型力量 x 120 lb力量 x 240 lb

战斗骰点

先攻

先攻加值 Initiative Modifier = 敏捷调整值

先攻判定 Roll Initiative = D20 + 先攻加值

物理攻击

近战攻击 Melee Attack = D20 + （熟练加值) + （力量）调整值 VS 护甲等级

远程攻击 Ranged Attack = D20 + （熟练加值) + （敏捷）调整值（投掷则为 力量） VS 护甲等级

擒抱 / 推撞 Grapple / Shove = 8 + 熟练加值 + 力量调整值 VS 力量 / 敏捷 豁免检定

法术攻击

法术攻击 Spell Attack = D20 + 熟练加值 + 施法属性调整值 VS 护甲等级

豁免难度 Difficulty Class（DC） = 8 + 熟练加值 + 施法属性调整值 VS 豁免检定

防御

护甲等级 Armor Class（AC） = 10 + 防具加值 + 敏捷调整值

半身掩护 Half Cover：护甲等级、敏捷豁免检定 +2

四分之三掩护 Three-Quarters Cover：护甲等级、敏捷豁免检定 +5

维持专注 Concentration = 10 / 伤害 / 2 (Max = 30)

伤害

非法术伤害 Damage = 伤害骰 + （属性调整值）（情况复杂，请当下判断）

坠落 Fall = 1d6 / 10 ft （Max = 20d6 ）

燃烧 Burn = 1d4 / 回合

跳高 / 跳远

跳高 High Jump = 10 ft 助跑 3 + 力量属性值 ft；无助跑减半；伸手触及 + (1.5 x 身高)

跳远Long Jump = 10 ft 助跑 力量属性值 ft；无助跑减半
