## 🎬 **跑团是什么？**

**跑团就像一群人一起拍电影**  

您是**主演**，我是**专属导演+编剧**，故事由 ==您的选择== 决定结局

您不是在看故事，而是在和朋友们 ==一起编写故事==

:::

---

- **主持人（DM）是您的导游**  
  - 负责提供场景、控制NPC、裁决规则  
  - 目标是 ==帮助您的角色迎接挑战、获得成长==

- **规则是工具，不是枷锁**  
  - 只要大家 ==玩得开心==，规则可以随时调整甚至忽略  
  - 重点是 ==创意、想象力和合作==，不是死记硬背条文

- **故事是大家一起写的**  
  - 您可以随时提出“我想加入 XX 元素”或“我不希望出现 XX 情节”  
  - 故事可能会因为 ==角色的每一个决定== 而改变走向

- **角色是您自己的投影**  
  - 设计角色时，想想 ==自己生活中最在乎什么、最害怕什么==
  - 只有您 ==真心在乎这个角色==，游戏才能帮您投入

- **安全第一，快乐第二**  
  - 任何时候觉得不舒服都可以使用“安全词”（如`打断铃`）暂停  
  - 冲突是 ==角色成长的机会==，不是玩家之间的对立

## 🧳 **新手准备清单**

**记住这5点，就能轻松开始**

:::

::: card

### **1. 心理准备：命题的舞台**
- 跑团像 ==自由的第二人生模拟==，不需要固定路线
- 情感体验是核心：选择 ==您会在乎、会动情== 的角色
- 使用 `打断铃` 或安全词随时暂停，==安全第一==

:::

::: card

### **2. 沟通期望：让我了解你**
- **喜欢什么**：角色塑造/社交政治/逻辑解谜/战斗挑战？
- **不想遇到什么**：提前说明雷区，我会全力避开
- **想加入什么**：大胆提出元素，==Rule of Cool==：够酷就可能实现

:::

::: card

### **3. 角色设计：先人物后数据**
1. **最 ==在乎/害怕== 什么？**
2. **用一个词/一句话来描述 ==人生命题==，会是什么？**
3. **希望面临什么样的 ==挑战==？**
4. **==惯用什么==面对困难？**

:::

::: center

> **数据规则交给我**，您专注思考**角色的人生**

:::

::: card

### **4. 团队共识：在场一致原则**
- 只要 ==大家都同意、玩得开心==，规则可调整、故事可回退
- 冲突是 ==角色成长机会==，不是玩家对立
- ==游戏一定能进行下去==，死亡不一定是终点

:::

::: card

### **5. 实际准备：旅程的奢侈**
- 预留完整时间（8-10小时）
- 保持 ==开放、创意== 的心态
- 准备好享受 ==属于这群人的独特故事==

:::

## 🎭 **我的带团风格**

**核心承诺**：追求角色独特的故事，期待角色用自己的方式开创不一样的展开

在此前提下，我会**提供协助**，非常欢迎您在这里 ==尽情扮演并发挥自己角色的特点==

:::

::: card

#### **故事第一，规则第二**
- 关注的是 ==发生了什么==，而非规则条框
- 每场战斗都有故事意义，**不会为难度而难度**
- 是 ==共同完成故事== 的伙伴，不是对手

:::

::: card

#### **角色驱动**
- 符合角色特点时，协助完成 ==高光时刻==
- 无法塑造独特故事时，**增加合理阻力，挑战角色**
- 长团中 ==挖掘情感痛点==，定制个人故事线

:::

::: card

#### **您的选择优先**
- 模组改造为 ==开放世界模拟==
- 相比既定叙事，更鼓励创造自己的路径
- ==Rule of Cool==：够酷、够扮演、大家想要，就值得实现

:::

::: card

#### **选择自由，责任明确**
- 玩家选择之外的原因导致游戏无法继续，是 ==我的责任==
- 角色可以 ==自由做出选择、承担行为后果==
- 游戏一定能进行下去，且尽力使其值得玩味
- 失败也可以是故事；死亡也可能是复活旅途的开始

:::

::: card

#### **资源丰富，策略多样**
- 倾向于给予 ==过量资源==
- 相应提升难度
- 目的是 ==丰富策略选择==
