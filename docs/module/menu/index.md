## 🎭 **我的带团风格**

**核心承诺**：追求角色独特的故事，期待角色用自己的方式开创不一样的展开

在此前提下，我会**提供协助**，非常欢迎您在这里 ==尽情扮演并发挥自己角色的特点==

:::

::: card

#### **故事第一，规则第二**
- 关注的是 ==发生了什么==，而非规则条框
- 每场战斗都有故事意义，**不会为难度而难度**
- 是 ==共同完成故事== 的伙伴，不是对手

:::

::: card

#### **角色驱动**
- 符合角色特点时，协助完成 ==高光时刻==
- 无法塑造独特故事时，**增加合理阻力，挑战角色**
- 长团中 ==挖掘情感痛点==，定制个人故事线

:::

::: card

#### **您的选择优先**
- 模组改造为 ==开放世界模拟==
- 相比既定叙事，更鼓励创造自己的路径
- ==Rule of Cool==：够酷、够扮演、大家想要，就值得实现

:::

::: card

#### **选择自由，责任明确**
- 玩家选择之外的原因导致游戏无法继续，是 ==我的责任==
- 角色可以 ==自由做出选择、承担行为后果==
- 游戏一定能进行下去，且尽力使其值得玩味
- 失败也可以是故事；死亡也可能是复活旅途的开始

:::

::: card

#### **资源丰富，策略多样**
- 倾向于给予 ==过量资源==
- 相应提升难度
- 目的是 ==丰富策略选择==

:::

### **🔴命运-围绕特征的共创想象**

:::

::::: card

:::: center

FATE 是一款轻量化的、需要发挥想象力、主持与玩家共同即兴发挥的泛用规则

仅仅提供了互动的规则，并不与任何故事联系，也不规定固定的“特征”词条

主要围绕`特征`的==自定义创作与即兴引用==游玩

::: warning 非命运之夜

该 FATE 并非指代 Fate 系列作品

:::

::::

---

该规则==极其简单容易上手、也极其具有弹性可供自定义==

1. 角色、场景、物件等各类事物的特点都由`特征`概括

2. `特征`可以被创造、消除，也根据持久性有不同分类

3. 主持与玩家拥有`命运点`，可以主动使用或被动强迫引用`特性`

4. 引用的`特性`会为各类对抗提供增/减益，并通过骰子对抗决定成败

---

::: right

该规则非常适合用于：

规则过于复杂，但希望游玩某模组

没有合适规则，希望即兴自由创作

:::

---

:::: center

::: note 模组兼容

该规则通常与任何模组可相互兼容

但游玩上会更聚焦于角色特征

同时也是开口胡团的最佳规则

### **🔮DND 5R-剑与魔法英雄战棋**

:::

:::::: card

:::: center

龙与地下城是一款以==中世纪、西方奇幻==为根基搭建的规则

是 TRPG 的元老，有相当长的历史

---

该规则通过不同的`种族`、`职业`、`专长`等词条搭配

再加上`法术`与`魔法道具`等超自然要素

常常赋予角色==各种强大的能力==去表现自身，==成就多样的冒险故事==

因此更倾向于打造==英雄传奇史诗故事==

::: note 老派版本

5 版规则，尤其是 2024 的 5R 版本，进一步增强了玩家的能力

相比于此前的 3R，同类的 PF，甚至更老的 aDND 等旧版 DND

更加容易获取丰富而且强大的能力，更强调了英雄史诗的一面

:::

::::

---

该规则已有多个已经成熟知名的世界设定：
- 被遗忘的国度（主流 DND 作品中常见的世界设定）
- 灰鹰
- 龙枪
- 艾伯伦
- 浩劫残阳

::: info 博德之门

《博德之门》就是在《被遗忘的国度》的世界设定中展开的

:::

---

::::: center

该规则的互动主要分为两部分：

:::: card-grid

::: card

**扮演**

利用普遍适用的`技能`

使用`特性`词条、`魔法道具`与`法术`

完成各种功能性、超自然的事务

---

包括但不限于：

超自然侦测与反侦测的信息获取

易容、命令、暗示等社交玩法

---

乐趣在于==创意、新奇地利用特殊能力==

:::

::: card

**战斗**

利用地形、走位、`状态`等回合制战棋玩法

使用`特性`词条、`魔法道具`与`法术`

击败敌人或完成战斗目标

---

常常会需要考虑：

有限的资源与`回合`经济

不同的地形、`掩护`与`状态`

---

乐趣在于==充分、巧妙利用角色与场景特点==
