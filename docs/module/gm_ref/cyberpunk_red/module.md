## 每集原则

每集独立成章，**不固定模板**。根据歌曲气质自由调整：

- **一首歌决定氛围**：Morphine=慵懒如糖浆；之后的空战=高速紧张；各自不同
- **Deadwax 的目标**：每集一张唱片，但获取方式迥异
- **队伍成长独立**：角色个人线与每集剧情自由搭配
- **博弈核心**：每集NPC自带的欲望与困境构成博弈难题

## 2045 / 2077 兼容说明

两个时代共用 NPC 和场景框架，但**叙事前提不同**。详见各 S 文件末尾的 `## 2077 情景` 章节：

| 要素 | 2045版（已发生的） | 2077版（没发生的结果） |
|---|---|---|
| **叙事前提** | 强迁前夜。Arthur Remain 亲自敲三扇门 | 三十二年后的空白。Arthur II 替父亲走他没走完的路 |
| **Deadwax** | Arthur Remain，65岁，亲自带队 | Arthur Remain II，约35岁。在父亲遗物中发现三张杯垫 |
| **Aisling** | Cold As Ice 主人，约40岁。不敢敲门 | 已故。孙女开 **Mara's**。杯垫封在玻璃下——祖母留的 |
| **Gummy** | 前军用斯安威斯坦使用者，约45岁 | 约77岁（若活着），或在安置点，或在诊所。可能平静了，也可能失控过 |
| **Stitch** | 旧货铺老板，五十多岁。用算账代替感情 | 已故。孙子在 Kabuki 接手。停摆的钟还在——没人知道为什么 |
| **Bolt** | 修车铺老板，近七十。等了三十年 | 已故。孙子继承。吉他挂在墙上——两代没人调过 |
| **Dell** | 二十出头，冲动收黑账 | 约52岁，老街区仅存的记忆者。什么都记得 |
| **博弈核心** | 替不敢敲门的人去敲门 | 替没机会敲门的人去敲门——后代对话，填补空白 |
| **唱片/CD** | 数据崩溃后唯一干净信息源 | 对企业"历史修正"的物理反证 |

> **运行原则**：2045 是"理应发生的事"，2077 是"如果 2045 没发生"的平行推演。2045 的每个关键节点（敲门/交易/对峙/终幕）若未完成→2077 变成"寻找爷爷辈遗物"+"两个后代替他们完成对话"。
