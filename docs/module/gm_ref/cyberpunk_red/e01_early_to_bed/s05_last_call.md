# Last Call

> S05 —— 终幕：Cold As Ice
>
> **Last Call**——酒吧术语：打烊前最后一轮。酒保敲铃，把灯调亮。意思是：喝完这杯，该走了。

---

## 回到 Cold As Ice

> 霓虹还是坏的。**C♁ld As Ice**——第三个字母烧到现在。没人修。
>
> *(也不用修了。再过几个小时企业安保拆墙的时候一块拆。)*
>
> 铁门掩着。铰链换过，锁舌加过——Combat Zone边缘开酒吧的标配。
> 监控探头的红灯还亮着。CD 机还在放——Foreigner。Cold As Ice。
> 下一首：Hollies。Long Cool Woman。两首来回。放了一整夜。
>
> 酒大半装箱了。瓶子按高度排列——她还没搬走。杯子剩最后三只。
> 干净。在吧台上。等了一整夜的不是客人。
>
> Aisling在吧台后面。红棕发裹褪色蓝头巾。脖子上还是那片褪色簧片。
>
> 听见脚步。抬头——这次没先说"关门了"。

## 三样东西

> 吧台够长。三样东西排开。
>
> **录音带。杂志。硬盘。**
>
> 沉默的长度刚好够CD 机把Cold As Ice放完。下一首还没起。

### 录音带

```
Gummy的磁带。Branko吹的。标签最旧，没有日期。

Aisling拿起来。翻了一面。又翻回来。
没放进录音机。只是握着。
「我知道四楼有。好多年了。」
停顿。
「我不知道他给了别人。」
（她调查过。知道磁带上是什么。但她不知道Branko给磁带的原因——
  "睡不着的时候放。"那是Branko对一个陌生士兵说的话。
  不是对她说的。现在她知道了。）
```

### Vox-Text 杂志

```
2021年夏季号。Branko的封面。萨克斯横在膝上，看着镜头笑。

她翻到那篇专访。看了很久。
不是在看——是在读。读了不止一遍可能。
然后翻到扉页。空白。拿起记号笔。
写了一个字。没让任何人看写了什么。
（DM：她写的是"Mara"。她自己的笔迹。给Branko没写完的回信。）
```

### 硬盘 / Demo 录音

```
Bolt的旧硬盘。不是正式demo ——一段排练室的碎片录音。
Branko在等乐队。闲着，对着麦克风哼歌。

Queen——Teo Torriatte。哼得不完整，断在"let us cling together"。
然后自言自语：「我女儿十岁了。不知道她会不会喜欢Queen。」
录音结束。

Aisling听完。摸簧片的动作停在一半。
沉默了很久——不是哭，是呼吸变浅了。
「……他哼的是Queen。」
然后，很轻地——像在确认：「是Teo Torriatte。」
（她听过。小时候。Branko在厨房吹萨克斯的时候放过这张唱片。
  她不知道他记得。他不知道她记得。
  两个人都记得。谁都没说。）
```

> 不是旋律。是锚。

> 没人说话。也不需要。

## Deadwax 在此：Zippo

```
Arthur靠在吧台角落——S01他站的地方。同一个位置。
手在裤兜里。银Zippo 。

整个终幕他没怎么说话。跟往常一样。
但今晚他没在进门之前把烟掐了。
他点了一根。在室内。Aisling没拦。

（——他知道这个Zippo 是Branko的。从什么时候开始知道的？

如果Arthur选择告诉Aisling：
他从裤兜里拿出Zippo 。放在吧台上——三样东西旁边。
「你爸的。」

Aisling看着打火机。外壳磨得看不清刻字。
「……我知道。我在吧台上放了三年。」
（她知道Arthur拿走的。她没拦。）
停顿。
「你用着吧。他更喜欢别人用他的东西。」

如果Arthur选择不说：
Zippo还在他兜里。点完那根烟——把烟掐了。
掐烟的姿势比别人掐一根烟更郑重。像在决定什么事。
```

## 情景

- **时间**：深夜。CD 播完就天亮。或者说——CD 不播完天不会亮。
- **不干预走向**：Aisling听完demo 的反应、Arthur可能说出Zippo 也可能不——取决于S01-S04攒下的东西。不强制"完美结局"。
- **吉他**：如果玩家提到Bolt和吉他——Aisling会愣一下。然后问：「什么吉他。」她不知道Branko想转型做摇滚。这条线留到第二集或留白。今夜的重点是三样东西——吉他是明天的故事。
- **CD**：最终Aisling把Morphine CD 给了玩家。条件完成了。
