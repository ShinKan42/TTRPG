# Underclock

> S02 —— Gummy & 录音带
>
> **Underclock**（降频）：计算机术语。刻意将处理器速度降至标称以下以换取稳定性。反义词：Overclock（超频）。

---

## 街景

> 第四栋公寓。比周围矮一层——顶层在'23年冲击波里塌了，没修。外墙裂缝从三楼爬到屋顶。
>
> 一楼门厅：信箱被撬空。地上几张踩烂的强迁通知——日期是昨天。电梯坏了一辈子。拾荒的正蹲在墙角翻废纸——不是住户。
>
> 二楼拐角：有人刚走——楼梯间还飘着烟味，不是烟草。脚步声往下，渐远。墙上新喷的漆："欠债还钱"——没干透。字迹潦草，不是第一次写。
>
> 三楼：门缝里电视声——还在播，说明还没搬。楼道晾着衣服，滴着水。这层有人赖着不走。
>
> 四楼：走廊尽头。灯坏了一半。三个小孩——最大的大概十岁，举着扫帚当剑。另一个裹着破窗帘当斗篷，第三个蹲在纸板"城堡"后面，负责旁白。
>
> 「勇士们——前方就是巨人的巢穴！」
>
> 「我侦测到邪恶！」
> 「你侦测到什么邪恶，你侦测到门缝漏光。」
> 「那就是邪恶的光！」
>
> 巨人住在走廊尽头的门里。勇士们一次次冲锋——每次都在门前一米急刹车。那个年纪的胆子刚好够冲到那个距离。
>
> 尽头那扇门——底缝漏出暖黄灯光。萨克斯。一段低沉的即兴，不像在吹奏，像在跟自己聊天。
>
> 小孩听见脚步声，以为是巨人——披斗篷的那个看清是外人，失望地啧了一声。扫帚剑垂下。「不是巨人。」三人抱着道具跑光。纸板城堡留在走廊中央。

```

  「我搬来三年，没见他下过几次楼。吃的有人送。」
  → 半同情半怕。同情是因为知道他是"打废了退下来的"。

  → 她见过什么。不想在孩子面前说。

## Gummy

| 命题 | 一个知道怎么杀人的人，用尽全力不去杀——但他不知道"温柔"够不够活下去。 |
|------|------|
| 动机 | 证明自己还是人。不是武器。 |
| 恐惧 | 环境逼他变回以前的东西。强迁今夜——他在失去"慢下来"的证明。 |

**外貌**：巨大。不是胖——纯骨架和残余的军用肌肉。1.9m+，手能把人的头包住。但他拿任何东西——磁带、杯子、门把手——都像用指尖在读。不是没力气。是怕力气。

**声音**：意外地轻，意外地慢——从那个体格出来像走错了声道。每个词被称过。不是迟钝——是**他习惯了把一切都放慢**，包括自己说话。

**属性**：
```
INT 5   REF 9   DEX 7   TECH 4
COOL 7  WILL 8  LUCK -  MOVE 6
BODY 9  EMP 4
```

**技能**：
```
近战(徒手) 8        军用格斗术 8        射击(突击步枪) 7

忍耐 8              胁迫 5              急救 5
追踪 6              重武器 5            语言(英) 6、(街头语) 5
```

**武术**：军用格斗术（Military Combatives）——Militech特种反应连标准训练。不是比赛——是杀伤。
```
战技：
  ▸ 缴械击（Disarming Strike）
    命中后强制对手脱手。DV15。成功→武器飞落2m外。
  ▸ 骨碎（Bone Break）
    擒抱成功后发动。DV17。成功→目标肢体骨折，该肢体-2所有行动，直至治疗。
     Gummy只用过一次——在里约，之后没再用。
  ▸ 压制锁（Restraint Hold）
    擒抱状态下维持。每回合自动造成2d6淤伤（不致死）。
    Gummy最常用的——因为可以让人不动，不用杀人。
  ▸ 反击偏转（Counter-Deflect）
    被近战攻击时可消耗反应。DV 对方攻击骰。成功→偏转攻击到相邻目标或墙壁。
    Gummy的本能——但最后一帧会强制偏转。不打人，打墙。
      → 石膏板上那十几个拳头印是这么来的。
```

```

         或使用者主动激活（一个自由动作）。

激活效果（持续至手动降频）：
  ▸ REF +4（基础REF 9 → 13）——常人反应极限的2.5倍
  ▸ 先攻自动第一位——且在突袭轮仍可行动（即使被突袭）
  ▸ 每回合额外一个行动——额外的移动或攻击动作
  ▸ 弹道感知——可尝试闪避子弹：REF+Evasion对抗攻击骰，DV+2
     （常人无法闪避子弹，斯安威斯坦让神经快过音速）
  ▸ 移动速度翻倍（MOVE 6→12）——可沿墙短距奔跑，利用垂直面
  ▸ 0.3秒感知窗口——世界逐帧播放。能看到：
     弹壳在空中翻滚、对手扳机指节泛白、血从伤口涌出前0.1秒的肌肉抽搐。
     *DM描述提示：用慢镜头语言。不是"他开枪了"，
     是"你看见他食指扣下——簧片压缩——撞针落下——火光从枪口绽放"*

代价：
  ▸ 无法区分威胁等级。任何快速移动=潜在目标。
     队友突然抬手、路人跑过、门被风吹开——都是"需要拦截的矢量"。
  ▸ 神经不会自动降频。停不下来。必须手动服药（慢钟）关闭。
  ▸ 激活越久，恢复越慢。持续超过2回合→降频需要2颗慢钟（而非1颗）。
  ▸ 停药症状：6-8小时内神经重新加速。先视神经→什么都变慢。
     再听觉→虫爬声放大为脚步。再触觉→衣服如砂纸。
     24小时不服药→永久性神经损伤（REF-2，不可逆）。

残留意识：
  已有文献记载。长期斯安威斯坦使用者报告——
  "加速状态中有另一个判断。不是自己的。但很熟悉。"
  前用户在行动中阵亡。Gummy是这副植入体的第二个主人。
  加速时偶尔听到战术口令——已死队友的呼号。
  *（DM：Gummy在加速状态下偶尔会低语回应——
  "收到""掩护左翼""看到了"。他在跟死人说话。）*

玩家若尝试摘除：
  伊文氏神经融合综合征——DV25 Cybertech手术。
  失败 → 永久REF-4 + EMP-3，可能终身瘫痪。
  成功 → 保留REF+1，可主动激活（REF+2，持续1回合，每日1次），
         但睡眠时强制性闪回——随机战斗记忆，醒来SAN-1d3。
```

## Gummy & Branko

```
2023年。战争最后几个月。Gummy随队在南部工业区执行牵制任务。

一次休整期——48小时离队许可。他无处可去。
走到Combat Zone边缘，听见萨克斯。
地下爵士酒吧——没招牌，门口一盏红灯。里面十几个人，
不是来听音乐的，是来躲的。躲炮击、躲征兵、躲外面的一切。

台上：一个老头在吹萨克斯。对着墙吹。没几个人听。
吹了一整夜。

Gummy站在最后面。没坐——当兵的习惯。听完。不走。
老头收了萨克斯，从他身边过。看了他一眼。
没说"你受伤了"——说——
「你站着不坐。当兵的？」

Gummy说「以前是」。

老头放下一盘磁带。「睡不着的时候放。」走了。

那盘磁带上没有日期。Gummy后来给它贴了第一个标记。
标记含义：今天没杀人。今天有人给了我一盘磁带。

那个老头叫Branko。Gummy不知道他的名字——一直不知道。
几个月后——荒坂塔核爆。冲击波震塌了三个街区外的地下爵士酒吧。
Branko死在里面。Gummy的部队在同一周遭遇伏击——全队覆没，他一个人活下来。
被回收。白房间。退役。

他搬进Combat Zone这栋公寓——因为便宜。因为没人在乎隔壁住谁。
几年后——楼下开了家酒吧。名字叫Cold As Ice。
有时候能听到萨克斯。不是同一个人吹的。但有时候是同一盘磁带里的旋律。
他不知道那酒吧是Branko的女儿开的。不知道那盘磁带是同一盘录音。

Aisling知道楼上的退伍兵手里的录音带是她爸的。
她调查过。知道那是父亲的录音——但她没去敲门。
十岁那年学的：在乎=给人离开你的机会。
她可以远远地收集父亲散落的遗物情报，
但不敢面对任何一个拿着父亲遗物的人。
```

## 录音带的意义

```
对Gummy：这盘不是他的"没杀人"日历上最早的那盘——Branko那盘是。
         最早的那盘没有日期——因为那时候他还没学会算日子。
         每晚听一遍萨克斯，等于听一遍"今天没杀人"。
         交出去=少了一晚的证明。
         不交=柜子里还有六盘日历，够今晚。但不够明晚。
