# Reverb

> S04 —— Bolt & 修车铺
>
> **Reverb**（混响）：声波撞上表面，弹回来——延迟、变形、但仍然是同一个声音。三十年后回到耳朵里的，已经是另一个东西了。

---

## 街景：修车铺

> 街对面。卷帘门拉了一半。灯光惨白——不是霓虹，是工作灯。
>
> 招牌：**BOLT**——没少灯管。每个字母都亮，但其中一个在闪。
>
> *(他会修东西。连招牌都修好了。)*
>
> 门口堆着几箱零件——打包好的。强迁准备。里面还在响：扳手拧螺丝，金属碰金属。收音机在放足球——伦敦口音的解说，信号不好，断断续续。
>
> 工作台上趴着一台拆了一半的引擎。旁边：扳手按大小排列。油腻的布叠得方正。
>
> *(他跟Aisling相反。他会修。他会整理。他会负责。他唯独不会开口。)*
>
> 墙角——吉他盒。黑色。贴着褪色的航空标签。标签上手写：Branko，c/o Bolt Sound.

## Bolt

| 命题 | 一个被欠了三十年的人——等的不是还，是承认。 |
|------|------|
| 动机 | 今晚离开。但如果Arthur进门——他可以多待一分钟。 |
| 恐惧 | 不是面对Arthur。是Arthur进门了却什么都不说。 |
| 默认行为 | 修东西。如果玩家搭话——手上不停。如果Arthur进门——可能会停。 |

**外貌**：近七十，跟Arthur差不多。方脸，灰白短发剃得短——不像发型，像习惯。厚框眼镜用医用胶带缠了一条腿。工作服——胸口绣着"BOLT"，字母是自己缝上去的。手：关节粗大，指甲缝里有机油。右手无名指有条绷带——不是伤口，是关节习惯了。

**声音**：伦敦东区——跟Arthur一样，但更亮。Arthur是砂纸，他是钢板。不拐弯，不问第二遍。句子结尾有小幅度上扬——不是问句，是习惯了跟机器说话。

**习惯**：手上永远有东西。说话不抬头——不是不尊重，是对着手里的零件说。扳手转三圈，扶一下眼镜——胶带又翘了。从来不换胶带。

## Bolt 个人史（DM参考）

```
Bolt——本名Robert Bolt。别人叫他Bolt不是因为姓——是因为二十岁那年
他在东伦敦开了Bolt Sound录音室。招牌挂上去，名字就定了。

他说自己是"老实修东西的"。修录音设备。修乐器。修引擎。
修东西比跟人打交道容易。机器坏了你知道哪里坏了。人坏了你看不出来。

2000年代——Bolt Sound是东伦敦小录音室里最稳定的。
不是最好的设备。但Bolt准时。设备校准了才开工。
Arthur的乐队每周三晚上来。不是最好的乐手——但准时。Bolt喜欢准时。

Branko在隔壁间录爵士。有时会过来，拎着萨克斯盒子，靠在门口听摇滚排练。
他说"伦敦的爵士太正统了"——然后问Bolt能不能帮他订把电吉他。
意大利，单板。Bolt垫了订金。他知道Branko不会赖账。

2015年——三个人分道扬镳。
Branko去了夜之城。琴到了——意大利寄了半年。
Arthur散了。欠三个月租金走的。吉他尾款也挂了。
Bolt把琴收进盒子。航空标签填了Branko的名字。没寄出去。
说不清为什么。可能是邮费太贵。可能是他还在等。

后来录音室也关了。欧洲经济危机。独立厂牌死一片。
企业招技术工——"有机械经验优先"。他把扳手装箱。上了飞机。
四十多岁从头开始。在夜之城开了修车铺。

某天在对街看到一个背影。灰色头发。老了。走路姿势没变。
Arthur。Bolt没喊。追了又怎样。欠的不是钱——是"为什么一声不吭走了"。
钱可以不要。但等了三十年没等到答案。

至于那把吉他——琴弦还是新的。没调过音。
不是纪念品——是证据。证明那个时代真的存在过。
证明Branko真的想转型做摇滚、Arthur真的每周三准时来排练、
Bolt Sound真的开了十五年。
明早这间铺子不存在了。这把琴去哪——Bolt没想好。
可能是烧了。可能是带走。可能是放在门口让第一个路过的人拿走。
他只确定一件事：不会卖给任何人。
```

**★2077**：Bolt 于 2060 年代在夜之城去世。修车铺传给了孙子——意大利电吉他还在墙上，琴弦还是新的。"爷爷说这把琴不卖。没说过为什么。"

## Bolt & Stitch

```
Bolt不知道Stitch的老婆怎么死的。只知道某天Stitch来修车铺时
眼睛是红的。Bolt没问。他从来不喜欢"节哀"——那种话说出来
不是你舒服，是说的人舒服。

所以第二天他照常去旧货铺收旧电器。什么都没发生。
Stitch照常报价。Bolt照常还价。一切照常。

后来发现墙角多了个纸箱。上面贴了标签，写着他的名字。
Stitch的字。整整齐齐的零件。标好了型号。
Bolt没问。下次来的时候放了一套修好的旧收音机在Stitch柜台上。
也没说。

两个男人花了好几年把一件天大的事压缩成塑料零件和旧收音机。
这是他们共同的语言。没人教过。
```

## Arthur 面对 Bolt

```
Arthur知道今晚要进那扇门。他还没决定。
如果玩家推他——他会看一眼卷帘门底下漏出的工作灯光。
「三十年了。」不是对任何人说的。

进去之后站着——不坐。Bolt不抬头。
「——比预估早了五十年。」
Arthur没接。沉默的长度刚好够一台引擎转三圈。
「你头发没了。」
「头发没了比你容易认了。」

（之后——看玩家。这是两个老男人的对话，但玩家在场。
 有人推一把→继续。没人推→可能就这么散了。
 Arthur欠的不是钱。是不能说的东西。
 Bolt等的不是还。是Arthur愿意说。）

如果玩家打破僵局（Cool+Empathy DV15）：
Bolt放下扳手。第一次抬头直视Arthur。
「你欠我的不是钱。你知道。」
Arthur的手指在敲大腿——旧日节拍。
「……知道。」
沉默。
Bolt从墙角拿起吉他盒。放在工作台上。没打开。

如果Arthur自己开口：
不需要检定。他会走到吉他盒前。
「这把……Branko的。」不是问句。
Bolt不回答。也不需要回答。
```

## Deadwax 在此

这是Arthur的战斗。不是玩家的。

从S01到S03，他一直是队伍后面那个不说话的人。
看Gummy。等Stitch。不进去。

现在轮到他了。

玩家可以劝他进去（Persuasion+Cool DV17）——
他会进去。一条命都走了七十年，不差这扇门。
但进去之后的沉默要他自己打破。没有检定可以替他做这个。

如果玩家不推——他可能站在卷帘门外，抽完一根烟，掐了，

## 情景

- **时间**：入夜。Bolt明早回英国。强迁前夜最后一个活。
- **不干预走向**：Arthur可能进去可能不进去。进去了可能开口可能不开口。玩家不是旁观者——是助推、是证人。不替代Arthur。
- **吉他**：不会交给玩家。结尾Bolt可能把它交给Aisling——等终幕决定。
- **Branko的录音**：Netrunning 获得。一段排练室碎片——Branko哼Queen，自语"我女儿十岁了"。玩家可带回S05给Aisling。

## 博弈

```
Bolt
  │  等了三十年。等的不是钱。是Arthur说"我欠你的"。
  │  如果Arthur说了——Bolt不需要更多。
  │  如果Arthur没说——他也不会生气。他等了三十年，不差再多等一晚。
  │  但他明早就走了。今晚是最后一晚。
  │  Bolt知道吉他意味着什么——一把没调过音的琴，
  │  锁着Branko的摇滚梦和Arthur的不告而别。他不知道自己守着两件事。
  │  他只知道一件事：这把琴不该跟他上飞机。

Arthur
  │  三十年回避。不是因为钱——是因为"解释不了"。
  │  为什么一声不吭走。为什么没回来说一声。为什么那批母带毁了。
  │  他说的可以是任何事——只要是真的。Bolt不一定原谅。
  │  但Arthur要的不是原谅。是开口。
  │  玩家在S01-S03攒了三个场景的信任。现在用上了。
  │  如果Arthur跟玩家已经建立了默契——他会更容易走进去。

玩家
  │  劝Arthur进门（DV17）。打破僵局（DV15）。
  │  帮Bolt修引擎——边拆边聊，对话自然过渡。
  │  Netrunning 挖出Branko哼Queen的录音——给终幕埋钩子。
  │  这不是战斗。但同样是博弈：推还是不推。替Arthur说还是让他自己说。
  │  替他说——他可能松了口气，也可能觉得被剥夺了开口的权利。
  │  不替他说——他可能永远不说。

Deadwax
  │  从S01的不敢敲门，到S04的进了门。
  │  玩家帮他走了这条路。他回报的方式是——
  │  在终幕，把Zippo 的事告诉Aisling。或者不说。
  │  取决于这四扇门他学到了什么。
```
