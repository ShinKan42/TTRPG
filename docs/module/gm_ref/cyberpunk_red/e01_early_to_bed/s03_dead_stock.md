# Dead Stock

> S03 —— Stitch & 拾荒者
>
> **Dead Stock**（死库存）：零售术语——卖不掉的货，停在架子上落灰。时间过去了，东西还在。等着一个不会来的买主。

---

## 街景：旧货铺

> 半条街外。招牌挂歪了——「S T I T C H」少了一盏灯管，剩下五个字母歪歪扭扭亮着。橱窗里什么都有：旧电脑、烂军靴、落灰的CD 架、一把没弦的吉他。
>
> 推门——铜铃响。没人应。
>
> 店里像被按了暂停。
>
> 墙上挂满钟。十几只——每一只停在不同时间。3:17。9:42。12:08。最靠近柜台那只，秒针还在抖，但走不动。
>
> *(每一只钟都死在不同时刻。像在说：时间不是停了——是离开了。)*
>
> 柜台后面——一个瘦男人在写东西。圆珠笔。旧账本。不抬头。
>
> 「看完了告诉我，我报价。」

## Stitch

| 命题 | 一个会算账的人，用算账来避免再算感情的账。 |
|------|------|
| 动机 | 维持"不亏"的状态。不欠任何人，不让任何人欠他——至少不欠感情债 |
| 恐惧 | 有人试图打破他的计算体系。有人让他再次"亏"了感情 |
| 默认行为 | 报价→等回应→成交或沉默。不主动提供信息。如果玩家提Bolt，他不会接——但会看墙角那个零件箱一眼 |

**外貌**：瘦。不是不健康——是刻意的节制。五十多岁，亚洲血统，发际线后退，铁框眼镜。工作衬衫袖子卷到手肘，前臂有旧烫伤疤——不是战斗，是焊东西。多口袋马甲，每个兜都有东西：笔、标签、卷尺、一盒没开封的图钉。

**声音**：每条句子等长。没有抑扬顿挫。不是冷漠——是*把所有话都换算成了价格*。说完就停。不追问。

**习惯**：写。一直在写——有人进店也在写。不是账，是清单。旧货铺所有东西都有一张手写清单——品牌、型号、收购价、心理价位。他的系统不用终端——在本子上。*终端会坏。纸不会。*

**属性**：

**技能**：

隐匿 5              手枪 5              语言(英) 8、(街头语) 6

**植入体**：右眼——基础光学增强（放大/测距）。不是战斗型——是他自己装的，为了看清旧电器上的标签和型号。左腕旧数据端口——还能用，基本不用。其他无。

## Stitch 个人史（DM参考）

```
Stitch——本名没人叫了。Dell叫他"舅"，顾客叫他"老板"，
Bolt叫他"Stitch"——唯一一个让他觉得这名字不刺耳的人。

四十岁之前：不一样的人。老婆叫Rhea——高个子，笑起来比他还大声。
在Combat Zone边缘开了这家旧货铺，她管钱，他管货。
生意不好不差——够活。够了。

2040年——洪水。工业区堤坝被企业炸了——转移泄洪方向，淹了工人住宅区。
Rhea那天在帮邻居搬家。水比通知来得快。
挖出来的时候，手表停在12:08。

Stitch没哭。至少没人见过他哭。
他把她的手表装进店里最旧的钟壳——那只钟现在挂在墙上，永远停在12:08。

从那天起——不给任何东西附加感情。不附加就不会亏。
这是他的"经营策略"。
也是他的病。

Bolt是老婆死后唯一没跟他说过"节哀"的人。
第二天照常来收旧电器。看了一眼墙上的新钟，什么都没说。
Stitch知道Bolt懂。Bolt欠了太多年的东西——他懂什么叫等。

所以他给Bolt留零件。不收钱。不解释。
Bolt也没问过。两个男人都不需要说话——这是他们共同的语言。

至于Aisling：
她来过一次。看到了玻璃柜里的杂志。
Stitch报了价。她没买。
没还价。没说什么——就是站着看了一会儿照片然后走了。
Stitch认出了那个表情。跟他在镜子里见过的一样。
他在等她再来。不是因为卖杂志——是因为他理解"没准备好"是什么感觉。
```

**★2077**：Stitch 于 2060 年代去世。他的孙子接手了旧货铺——但 Combat Zone 已被企业广场覆盖，铺子搬到了 Kabuki 地下市场。那十几只停摆的钟还在墙上。没人知道它们为什么停在那些时刻。也没人问。

```

## 情景

- **时间**：入夜。Stitch打烊了——但强迁前夜，开着门。明天所有东西都得搬走。他在做最后的盘点。
- **不干预走向**：Stitch开价→玩家可以交易、说服、撬开他。撬不开他会照常报价——不生气，不降价，不关门。他只是继续写清单。
- **公寓楼**：拾荒者据守车库。急救包在杂物间G。玩家可选择正面突入、潜行、Netrunning 、或喊话交易。环境互动物品提供战术选择，但不强制复杂决策。

## 博弈

```
Stitch
  │  明天强迁。二十年存货——钟、零件、杂志、记忆——要装箱。
  │  他的系统是"不附加感情"——但他附加了。
  │  Bolt的零件箱是白送的。老婆的手表在钟壳里。杂志等着Aisling回头。
  │  每个"不附加"背后都有一个"已附加"的证物。
  │  玩家能不能让他承认——哪怕只承认一件？
  │  如果玩家修好了那只抖动的钟——他报价时会扣50eb。
  │  不是感谢。是那只钟走起来让他想起了时间还在动。
