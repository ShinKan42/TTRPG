## **核心系统**
:::

:::: card

::: center
### **留声盒**
:::

- **身份识别**：==每个特工的唯一身份 ID==。

- **通话机制**：==双话筒==——话筒发起通讯，听筒接收中台语音。

- **通讯规则**：
   - ==中台不主动联系 `PC`==——特工执行任务时中台主动联系可能导致暴露。
   - 通讯必须由 `PC` 这一端发起。

- **使用条件**：
   - 发出声音 + 双手操作 + 发出微光。

- **转达消息**：`PC` 可通过留声盒让中台转达消息给联络人（若对方持有另一个留声盒）。

::: center
### **中台**
:::

首次激活投 <Badge type="danger" text="1d4"/> 决定哪个中台在线。

12h 内续钥匙 → 同中台可延长值班。

断连后重激活 → 再投 <Badge type="danger" text="1d4"/>——==可能换人==，可能撞回同一个。

代号 = 和弦音名串（如 Am7 = A-C-E-G）。
读法为 NATO 音标字母对应。

**中台接线员**：

- 每个单元剧独立设计 4 名接线员（1d4 表）。

- 接线员通过和弦的情感气质与任务基调匹配。

- 每名接线员包含：代号（和弦音名串）/ 读法（NATO对应）/ 声音特征 / 性格一句话。

**通讯规则**：==通信用最短语言==。
中台在收到联络后==提供任务所需信息，不主动超出==。

::: center
### **Sibylle 动机链**
:::

- **驱动**：==知识的武器化==——武器的垄断。
   - 她算的是算术：知识在一个人手里，这个人的选择决定了死多少人。
   - 知识在所有人手里——所有人同时选择，死得更多。
- **终极目标**：掌握邪恶之书（Book of Vile Darkness），==用自己的方式封存==。
   - ==不销毁==——销毁是放弃理解。
   - 封存是==垄断理解本身==。
- **为什么研究卵**：卵是可中和的远古邪物。水晶容器是一套==现成的封存机制==。
   - 但水晶中和会==毁掉样本==——变惰性 = 失去研究价值。
   - 她需要亲手杀死怪物并制成标本——==这些只有从完整尸体上提取==。
   - 数据不打算上交给 <Badge type="info" text="STEAL"/>。
