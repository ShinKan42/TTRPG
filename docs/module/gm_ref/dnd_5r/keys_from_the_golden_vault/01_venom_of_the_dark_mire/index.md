## **主题命题**
:::

> 博物馆里有一块石头——那是一颗卵。午夜就会孵化。
>
> 唯一知道真相的学者刚被开除。也只有她能中和它。
>
> 没人在乎。每个人都有自己的理由。

> **PC 在做的每一个选择在回答**：
>
> 你要为自己==不能百分百确定==的事——一个被否决的学者、一颗不知道会不会真孵化的卵、一座你偷了就可能毁掉的博物馆——==做到什么地步==？

### **丹内尔博士（Dr. Cassee Dannell）——失格学者**
:::

- **锚点台词**：「我可能错了。但如果我对了——今晚死的人不会原谅你们先花了一小时确认我有没有疯。」

- **身份**：
   - 前博物馆研究员。昨晚潜入想偷卵——被保安抓住。
   - 今天被开除。

- **她想要什么**：
   - 卵被中和。不是保护——是**以防万一**。
   - 她不能百分百确定卵会孵化。但她确定如果孵化了，没人知道怎么杀那个东西。

- **她会问 PC**："你们见过石头的脉动吗？用拇指按在上面。
   等三秒。
   你感觉到的是心跳——不是你的。   "。

- **价值观**："我可能错了。
   但如果我对了——今晚死的人不会原谅你们先花了一小时确认我有没有疯。
   "。

- **对抗手段**：
   - 博物馆结构图、她知道的部分警报位置。
   - 没有这些——PC 对馆内几乎一摸黑。
   - 她给情报，但 PC 得先让她觉得他们当回事。

- **底线被碰时**：如果 PC 敷衍她——她会自己去第二次。
   一个人。
   她知道上次怎么被抓住的。
   这次不会。
- **在等什么**（伺机）：等 PC 决定信不信她。

### 阿尔达·阿金（Alda Arkin）——馆长

:::

- **锚点台词**：「人心防备的大山，比大山底下的化石还要顽固。」（精灵语·法语：Les montagnes de la méfiance humaine sont plus obstinées que les fossiles sous les montagnes.）

- **外貌**：
   - 瘦削老精灵。眼镜链从耳后垂到锁骨。盘发紧得没一根是松的。
   - 声音尖、干、快。像教导主任——但不是刻薄，是太专注了以至于顾不上柔软。

- **身份**：
   - 瓦肯布拉夫大学考古博物馆馆长。位面考古学出身。
   - 年轻时在火元素位面做田野——回来后写了三本书。
   - 现在不常出手——但她的魔法和她的学术是一回事：旧、深、不可忽视。

- **她的执念**：
   - 考古。策展说明每个字都是她亲自写的。
   - "石头——看这纹路。不是地质的。是生物的。很久以前有东西在里面。"
   - 她写了这段话，但**不信<Badge type="info" text="丹内尔博士"/>说它还活着**。不是恶意——是真觉得丹内尔研究走火入魔了。

- **她的困境**：
   - 拍卖是她最讨厌的事。但她做了——为了不让理事会把博物馆改成商业画廊。
   - 这个矛盾她不对别人说。

- **三层算盘**（咬合角力——她不是在赌，是两边都压）：
   - **表层**：拍卖邀请函 100 gp/人 + 山珍海味餐饮赞助。人头费保底续命。
   - **内层**：石头卖给 <Badge type="info" text="STEAL"/> 当场成交。价格内定，有回扣，但也够用。STEAL 交割走<Badge type="info" text="装卸区"/>坡道，<Badge type="info" text="装卸工"/>执行。
   - **为什么走拍卖流程**：
      - 学院资产不能私下给——需要"合法"交易记录。
      - 拍卖 = 合法流程 + 合理价格 + 有见证人。
      - <Badge type="info" text="STEAL"/> 不确定收藏家会不会二次出手，需要尽快拿到并投放——拍卖是最快的方式。
      - <Badge type="info" text="Lorenzo"/> 举牌"代表自己" = 掩人耳目，但收藏家跟价反而帮了忙——显得是正常竞价不是内定。
   - **对冲层**：如果晚上孵化且不损坏馆内 → 消灭/控制怪物 → 拍卖钱财保住 + 展品安全 + 外墙返修基金 + 重建资助 + 保险赔偿。
   - **结构性方便**：她不信丹内尔——但她的不信是**方便的**。信了就得处理，不信就能继续卖。这不是认知失调，是多边下注。

- **她会问 PC**（白天勘探时）："你们看那块石头——觉得它是什么年代的？"（不是寒暄。
   她真的想知道。
   ）。

- **对抗手段**：
   - 控场魔法（捆、困、封——不是杀）、对博物馆的绝对统治。
   - 如果偷窃败露，她能同时面对 PC 和 <Badge type="info" text="Gareth"/>——都让人不想正面接她。

- **底线被碰时**：
   - 看到有人在她的博物馆里偷东西——不是先问，是**先出手**。
   - 然后站在被捆住的你面前，老花镜推到额头上。"说。"

- **在等什么**（伺机）：拍卖的结果。
   但更深层——她在等有人告诉她丹内尔是对的。
   她不会主动问。
   但她留了一个缝隙。

### **Ithréïne（伊丝瑞恩）——竞争队伍核心**
:::

- **锚点台词**：「代价我已经算过了。」（精灵语·法语：J'ai déjà calculé le prix.）

- **`种族`/`职业`**：`高等精灵`。`预言`学派。面纱遮眼不遮脸。

- **驱动**：==知识的武器化==——武器的垄断。
   - 她算的是算术：知识在一个人手里，这个人的选择决定了死多少人。
   - 知识在所有人手里——所有人同时选择，死得更多。
   - 她不在"对错"上内耗——她在==执行上博弈==。

- **终极目标**：掌握邪恶之书（Book of Vile Darkness），==用自己的方式封存它==。
   - 让它再也不被使用。
   - ==不销毁==——销毁是放弃理解。
   - 封存是==垄断理解本身==。
   - 书在谁手里谁就是==威胁==。
   - 在她手里、以她的方式封住——威胁归零，而理解保留在她一人手里。

- **为什么研究卵**：
   - 卵是可中和的远古邪物——==水晶容器能让它变惰性==。
   - 这是一套现成的封存机制。
   - 但水晶中和会毁掉样本——变惰性 = 失去研究价值。
   - 她需要亲手杀死怪物并制成标本。
   - 她要的是解剖结构、器官组织、脉动器官的生理机制、与远古邪书的共性。
   - ==这些只有从完整尸体上提取==。
   - 数据不打算上交给 <Badge type="info" text="STEAL"/>。

- **和 STEAL 的关系**：
   - 明面遵从 <Badge type="info" text="STEAL"/>，暗地做自己的独立项目。
   - <Badge type="info" text="STEAL"/> 要混乱要利润，她用混乱和利润掩盖自己的知识收集。
   - 目前两边满意，因为她的预言术是 STEAL 的不可替代资产。

- **今晚四方逐力**：
   - **STEAL**：要蛋完好带走当武器。
      - 但缺水晶容器和中和信息——注定失败。
   - **Ithréïne 的实计**：
      - 拖时间让蛋孵化。
      - 最好拖到 PC 也在场——PC 帮忙击败怪物。
      - 怪物死后**归属摊牌**：她要尸体做标本，PC 要水晶中和（保护生命和藏品）。
      - ==这是四方逐力的最终冲突点==。
      - 不主动杀人——杀人制造噪音，噪音威胁知识垄断。
      - 但 PC 若执意毁掉尸体，她会出手。
   - **<Badge type="info" text="馆长"/>**：要钱要保护博物馆。
      - 知道会孵化但不知是今晚。
      - 可能和 STEAL 交易——前提是她获得了足够信息。
   - **PC**：==实施盗窃==。
      - 之前各阶段是侦察，现在是执行。
   - **她需要的证物**：
      - PC 的痕迹——<Badge type="info" text="黄金宝库"/>的钥匙碎片、一个被目击到的面孔、一个可以指认的物证。
      - 用来说"他们先到了。我们被拦截。不是我们的失职。"
      - ==PC 是活着的替罪羊==——越活跃越好。

- **为什么不自己动手**：
   - Rival 需要的不是拿走卵——STEAL 要的才是活卵。
   - Ithréïne 要的是**杀死孵化后的怪物并制成标本**。
   - 活卵交给 STEAL = 无法孵化 = 无法击杀 = 没有标本。
   - 所以 Rival 必须等卵孵化，而孵化需要卵留在馆内到午夜。
   - 他们不能提前拿走——提前拿走卵 STEAL 就交割成功了。
   - <Badge type="info" text="Gareth"/> 有合法理由接近卵但不会拿——拿了等于帮 <Badge type="info" text="STEAL"/> 交割，毁了 Ithréïne 的标本计划。

- **对抗手段**：预言 + 魔宠 + <Badge type="info" text="Kaël"/> 的跟踪。
   - 她知道 PC 的下一步——不是全知，但够多。
- **底线被碰时**：==被看见==。
   - 面纱后的眼神被捕捉到一次——她就走了。
   - 下次回来是另一个人。
- ==不出现==。
   - 灰鸦在。
   - 她不需要亲自来——来了是给 PC 机会认识她。
   - 她不会给。

### **邪恶之书（终极目标）的侵入机制**
:::

- 书是==物理性腐败源头==——**被动侵蚀**周围物质。
   - B30 旗石烧焦龟裂、墙上细缝延伸进黑暗。
   - B29 焦痕和碎裂砖石从半位面门缝渗出
- 连 Vrakir 的半位面都封不住——每次开门，腐蚀就渗出来。
- 不是精神控制（虽然也有那面），核心是**对物质的持续侵入**——改写周围物质的现实结构。


::::
:::: card
::: center
### **怪物血肉 = 物理屏障**
:::

- 书侵蚀物质 → 怪物血肉不断适应、吞噬、**长出新组织** → 再生速度超过侵蚀速度 = 物理阻挡。
- 怪物被酸/腐蚀/暗蚀打中后，额外长出保护性组织（临时 `HP`）。这是**抗腐蚀生物材料**。
- 伊瑟瑞恩要的：尸体上的适应血肉 → 提取保存 → 物理包裹邪恶之书 → 阻挡书的侵蚀。
