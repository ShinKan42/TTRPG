## **丹内尔请求**
:::

::: note 立场问题

你要为自己==不能百分百确定==的事——一个==被否决的学者==、一颗不知道会不会真孵化的卵——==做到什么地步==？

每个选择都在回答这个问题。

:::

::: center
## **丹内尔住处**
:::

::: note 氛围

- 博物馆后三条街，矮楼。楼下洗衣房，蒸气溢到街上。
- 上楼——洗衣粉味里掺进旧书页。干的，翻过很多次的老纸。
- 门没合上。椅上搭着昨晚外套，袖底褪色污渍，泥还是血。
- 桌上铺满。手绘平面图。水晶容器空的，在桌角。
- 茶杯冷，杯口细茶渍。
- 左手腕青了一块——被拧的。倒水想遮，袖口短了半寸。水壶在抖。
- 平面图宝石侧厅圈了又圈。
- 底下压张纸，划掉一行字，很浅——「==学术贡献无法核实==」。

:::

::: center
### **闲笔**
:::

- 桌角纸片潦草写「10.5h」
- 茶杯三小时前泡的

::: note

**节奏**

- **笑点**：茶冷三小时、倒水遮淤青袖口短半寸、桌角潦草「10.5h」。
- **沉默**：地图下压着划掉的「学术贡献无法核实」。

:::

## **丹内尔博士**
:::

- **自我**：人类`预言`学派 `法师`。
   - 学术圈最年轻的终身教授——==在被开除之前==。
   - 她的学科是驱魔法阵应用考古——用`防护法阵`验证文物的魔法活性。
      - 同行说是「==江湖术士的幌子==」。
      - ==她的数据没错==。但她在考古系里用魔法——在费伦的考古系里。

- **名字含义**：<Badge type="info" text="丹内尔博士"/>(<Badge type="info" text="Dannell"/>)=「山谷中的」(`通用语`)。
   - 祖辈家训：做山里那样朴实率直的人。
   - 这也是她倒霉的根源——==她太直了==。

- **人味**：
  1. **给每件出土文物起昵称**——「这个歪底的罐子叫老跛子，因为它站不稳。」
     - 没人记得编号，==所有人都记得老跛子==。
  2. **说话时无意识地用学术论文句式开头**——「基于上述观察……」「考虑到对照组的数据……」
     - 朋友跟她聊天气，她答「从长期趋势来看，降雨概率在上升」。
  3. **着急时咬铅笔尾端**——咬到木头毛了才换一支。
     - 桌上散着三四支咬烂的铅笔。
     - 笔迹从清晰变成潦草再变回清晰——==每次换笔都是一次深呼吸==。

- **状态**：昨晚潜入博物馆被抓。被开除了。
  - 手腕青了。茶冷了三个小时。
  - 笔记铺了一桌——铅笔叠墨迹，新字写在旧字上面。

- **动机**：卵必须被中和。
  - 她不能百分百确定——但她确定如果孵化了，==没人知道怎么杀那个东西==。
  - 别人说她疯了。==她习惯了==。

- **内核**：
  - 被开除、手腕被拧、被说成江湖骗子。她还在画地图，手指停在宝石侧厅那个圈上。
  - ==怕归怕，手没停==。
  - 三十五岁的人，七十岁的论文量。
  - 学生说「她是最好的老师」。同事说「她的方法论不科学」。
  - 她的回答是昨晚一个人去偷卵。被抓住了。
  - 今晚她找了帮手。==没放弃，换了个方式==。

:::

一人将钥匙插入留声盒。极低沉的机械音。

**宣读**（投 <Badge type="danger" text="1d4"/> 决定哪台中台）：

「我是中台。呼号 A-C-E-G。代号 Am7。时效十二小时。任务目标——黯沉泥沼石。时限——午夜前。联络人——<Badge type="info" text="丹内尔博士"/>。任务开始。」

钥匙弹出。留声盒进入活跃状态。

首次激活投 <Badge type="danger" text="1d4"/>，**断连后重激活再投**。

- ==中台不主动联系 PC==——特工执行任务时中台主动联系可能导致暴露。通讯必须由 PC 这一端发起。

## **Kaël — 对面茶馆**
:::

- **自我**：半精灵`游侠`。前赏金猎人——最好的那种。
   - 杀人不是他最多，==找人他最准==。
   - 话少。每个字都该有用处。
   - 端茶、放杯、偏头看鸽子——没有多余动作。

- **名字含义**：<Badge type="info" text="Kaël"/>=「忠贞的/忠诚的」(`精灵语`·法语)——==讽刺==。他忠于的只有那本小本子。

- **人味**：
  1. **在小本子空白处画简笔画**——不是预言碎片，是想象出来的怪兽。
     - 画完了就撕掉。
  2. **挑座位像选伏击点**——进任何空间，先找出谁能看见他、谁看不见他。
     - 坐下后第一眼不看桌子，看所有人的手。
     - 不是不信任，是他在野外活太久，**城市里的墙让他不安**。
  3. **收弓前有一道固定仪式**——收弓入袋前，用拇指沿弓背从弓梢滑到弓腹再滑回来。
     - ==不是检查，是指尖确认它还在==。
     - 没有人教他这个动作。

- **状态**：茶馆二楼靠窗。面前一杯茶没碰过。
  - 在等 PC 进去——也在等 PC 出来。
  - <Badge type="info" text="黄金宝库"/>的联系是他投递给<Badge type="info" text="丹内尔博士"/>的。
  - 他想看那封信引来了什么样的人。他来确认。

- **内核**：
  - 有个小本子，写满了 <Badge type="info" text="Ithréïne"/> 没说出来的预言碎片。
  - 他知道她比她说出来的知道得更多。不质疑。但记着。
  - 茶馆窗边这个人，==不站任何一边==。
  - 坐在窗边，看你们进楼，看你们出来。

### **对话**
:::

**走过去**：不躲。手从桌上拿开——让你看到没拿武器。

「你们就是<Badge type="info" text="丹内尔博士"/>在等的人。」陈述。

**问：是谁？**：「只是想看看那封信引来了什么样的人。」

站起来。放两枚硬币——一枚茶钱，一枚给服务生。杯子满的，没碰。

「我还有事。祝你们顺利。」走。不回头。

**问：什么信？**：停了半拍。没转身。

「她不认识我。我做了我觉得对的事。」继续走。

**追/拦**：不反抗。停下。回头看你——脸上没有愤怒，==只在等你自己想明白==。

::: note PC张力

- 追不追他？拦下来他不说。
- 一边「问清楚」，一边「==多一事不如少一事==」。
- `PL` 会记住「我们放过了一个知情者」。

:::
