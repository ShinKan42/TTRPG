:::

**宣读——<Badge type="info" text="馆长"/>排练片段**（`PC` 若溜进/阳台俯瞰听到）：

"……这块石头。看这纹路。不是地质的。是生物的。很久以前有东西在里面。今晚它将找到新主人——一位……（她停了。看石头。眼镜推上去。又拉下来。继续。）一位懂得其价值的收藏家。起拍价——"

==她没念出起拍价==。看了石头一眼。走了。

::: note 节奏材料

## **新手易漏**
:::


- **邀请函价格本身是情报**： **100 gp** 告诉 `PC` 混进去解决不了问题——要么偷、要么有贵人带、要么不进

- **17:00–18:00 无 `警报术`**： ==20:00 才激活==。这窗口溜进关闭展区有风险（巡逻）但不会触发魔法警报

- **<Badge type="info" text="Gareth"/> 不在岗**： ==17:00–18:00 他在换装==。这窗口馆内安保是副手带队——不是 Gareth 本人在判断

- **<Badge type="info" text="馆长"/>在 V13 排练**： ==17:30–17:50 她独自在宝石侧厅==。这窗口宝石侧厅有人（她），里面是馆长本人，不是安保

- **签到员不核对脸**： ==她核对名字不核对脸==。冒充关键不在像不像，在名字对不对

- ==密道不是 `PC` 独占==： ==<Badge type="info" text="Sibylle"/> 知道==。==从密道进=被观察==。免费出口不免费

::: center
## **`PC` 张力**
:::


- ==馆内躲藏者== vs ==馆外重入者==——信息不对称。馆内的人不知道 <Badge type="info" text="Gareth"/> 何时换好装上岗；馆外的人不知道<Badge type="info" text="馆长"/>在 V13 排练
- ==溜进宝石侧厅看馆长排练== vs ==避开她==——见到她人不在时的石头状态（脉动是否变化？）
- ==留声盒联系丹内尔== vs ==安静角落暴露位置==——丹内尔在搬东西能接但不便说话
- ==返回<Badge type="info" text="丹内尔博士"/>处可能再遇 <Badge type="info" text="Fidèle"/>==——他夹中，阻挠丹内尔+准备狩猎

::: center
## **`PL` 乐趣**
:::


- `PC` 分头行动时各 `PL` 掌控自己视线内的信息
- 馆内 `PL` 知道馆长排练的细节；馆外 `PL` 知道 Gareth 进员工门的时点。两边对不上=真人博弈
- 邀请函怎么搞——`PL` 自己想。规则只提供客观材料
