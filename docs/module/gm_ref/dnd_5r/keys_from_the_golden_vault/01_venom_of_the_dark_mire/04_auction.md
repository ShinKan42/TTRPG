## **拍卖舞会**
:::

::: note 立场问题
在香槟和琴声里，每个人都在笑着试探。

你说的每句话都被当成情报。

你的每个反应都被当成弱点。

你要怎么在==不出手的情况下==，让自己==不被看穿==——或者，你==要不要被看穿==？
:::
