## **开始之前**

赛博朋克 RED 的车卡不是填表。是你和 DM 一起**搭建一个人的底层逻辑**——他从哪里来、靠什么活、付出了什么代价。

规则书很厚。作为玩家，最重要的工作不是算数——是**决定你要扮演谁**。DM 会协助你处理数据计算、义体联动、技能检定细节。

**RED 的核心只有两层博弈：人性，和经济。** 人性能不能守住——你每装一件义体都离自己更远一寸。钱够不够撑到下个月——夜之城从不等你准备好了再收租。这两层博弈贯穿车卡全程，也贯穿每一场游戏。

## **一、时代的问答**

:::

在碰任何数字之前，先回答五道题。不需要写很多——一两句就够。但这些答案会伴随你的角色走过整场游戏。

:::: card-grid

::: card

::: info 你怎么活下来的？

夜之城每天都在筛人。你靠什么撑到今天——暴力、技术、人脉、运气，还是听话？

:::

:::

::: card

::: info 你有过什么，失去过什么？

失去的可能是人、身体的某一部分、某种信念、或者一个本来可以不一样的未来。

:::

:::

::: card

::: info 你现在想要什么？

钱不是答案——钱只是手段。你要什么？尊重？自由？报仇？一个能闭上眼睛的地方？

:::

:::

::: card

::: info 你愿意用什么去换？

人性、身体、良知、承诺、记忆——你准备卖掉哪一样？你给自己划的红线在哪里？

:::

:::

::: card

::: info 你住在哪里，怎么吃饭？

这个问题很具体——因为夜之城的具体问题最要命。集装箱还是公寓？合成糊还是真米饭？**你的每日开支就是你的自由程度。**

:::

## **二、生命之路**

:::

生命之路 (Lifepath) 是 RED 帮你发现自己是谁的工具。你不是在给自己编履历——你是在**挖掘一个已经活过二三十年的人**，弄清他爱过谁、恨过谁、为什么成了今天这副样子。

从以下五个维度，逐一选择最有感觉的条目。不要想"这个有没有用"——想**"这个人应该是这样的"**。

| 维度 | 你在决定什么 |
|---|---|
| **文化起源** | 出生在哪里？说什么母语？家庭来自什么文化背景？ |
| **人格特质** | 性格底色是什么？信任别人吗？穿衣服是为了什么？ |
| **家庭背景** | 父母是否健在？关系如何？在街头长大还是在企业家属区？ |
| **动机** | 什么推着你走？钱、复仇、爱、证明、还是单纯无处可去？ |
| **人生事件** | 有哪些重要的人——朋友、敌人、旧情人、未清算的账？ |

::: tip 如何对待生命之路

1. 参考：在 RED 核心规则书的生命之路表中选择与你的角色构想最契合的条目
2. 你不需要全部记住。但**你的动机和那几个重要的人**——是你每次做选择时都要知道的坐标
3. 生命之路的核心不是"提供背景信息"——是**让你自己在车卡阶段就搞清楚：我要扮演的是一个什么样的人**

:::

每个角色有一项专属**角色能力 (Role Ability)**，等级 1-10。别人可以通过技能模仿皮毛，但偷不走。

## **四、你的身体，你的代价**

:::

RED 有 **10 项属性**。每项从 2 起步（免费），未经义体改造最高 8。你有 **62 点** 可自由分配，每提升 1 点消耗 1 点。

属性的作用是决定**技能基础值**：

> **技能基础 = 属性值 + 对应技能等级**

属性越高，你能把对应技能推到的上限越高。

### 十项属性

:::: card-grid

::: card

#### **心智**

| 属性 | 掌管 |
|---|---|
| **智力 INT** | 逻辑、记忆、推理、知识。教育技能群的主属性。 |
| **反应 REF** | 手眼协调、出手速度。**所有远程武器的核心属性。** |
| **沉着 COOL** | 压力下的镇定。社交技能群的主属性。 |
| **技术 TECH** | 动手、修理、理解机器的直觉。技术技能群的主属性。 |
| **意志 WILL** | 精神坚韧、抗压、耐力。**直接影响 HP 计算。** |

:::

::: card

#### **身体**

| 属性 | 掌管 |
|---|---|
| **敏捷 DEX** | 身体精准控制。近战与潜行的核心属性。 |
| **移动 MOVE** | 跑速、游泳、闪避范围。**决定每回合移动距离 = MOVE × 2 米。** |
| **身体 BODY** | 蛮力、抗打、近战伤害。**直接影响 HP 计算和死亡豁免。** |

:::

::: card

#### **灵魂**

| 属性 | 掌管 |
|---|---|
| **同理心 EMP** | 读写情绪、建立连接。**决定人性值上限 = EMP × 10**。安装义体会扣减人性值，扣多了就不再是自己。详见[人性与金属](#七你身上有多少还是你的)。 |
| **运气 LUCK** | 不绑定任何技能。你有一个等于 LUCK 值的**幸运池**——每次消耗 1 点 = 任意检定 +1。池子消耗后在整个团次内不恢复（除非 DM 允许特定事件回充）。**高 LUCK 不是命好——是敢赌。用完之前你是底牌最多的人。用完以后你就没有安全网了。** |

:::

::::

::: info 属性值速查

| 值 | 含义 |
|---|---|
| 2 | 严重缺陷 |
| 3-4 | 低于平均 |
| 5-6 | 普通人水平 |
| 7 | 优秀。比多数人强一档 |
| 8 | 人类巅峰。再往上只能靠义体 |

:::

### 分配心法

不要先想"我要加什么"——先想**"我能接受什么很低"**。每一项属性偏低都有后果。

:::: card-grid

::: card

**INT 低** → 教育技能群整体上限受限。很多事你不知道——不是不想知道。

:::

::: card

**REF 低** → 所有远程武器命中偏低。你不是最快的枪。

:::

::: card

**EMP 低** → 人性池浅。装不了几件义体就到极限。读人也读不准。

:::

::: card

**LUCK 低** → 没有临门一脚的底牌。关键检定时没有再加 1 的选项。

:::

::::

> **★2077**：2077 使用 5 属性制（Body / Reflexes / Technical Ability / Intelligence / Cool），EMP / WILL / LUCK / DEX / MOVE 被合并。桌面 RED 保留 10 属性制。

## **五、你会什么，你忘过什么**

:::

RED 有 **66 项技能**，每项绑定一个属性。

技能分配分两步：**先落定保底，再自由分配。**

### 第一步：保底

从生命之路的**文化起源**出发，你和 DM 一起确定以下最低技能：

| 保底项 | 来源 | 说明 |
|---|---|---|
| **母语** | 文化起源 | 等级由出身决定。在母国长大的通常 4 级（流畅读写）。 |
| **街头语** (Streetslang) | 夜之城生存必需 | 如果街头语不是母语，至少 2 级。不会街头语在夜之城等于聋哑。 |
| **通识** (Education) | 家教或学校 | 有没有受过教育、受过多少——取决于家庭背景。0 级就是文盲。 |
| **本地知识** (Local Expert) | 你的地盘 | 你在哪片街区长大，就了解那片街区。2 级起步。 |

> 以上保底所需点数从生命之路推导，不计入你的自由分配池。**保底是你已经有的——自由分配是你选择去学的。**

### 第二步：自由分配

扣除保底后，你有 **剩余技能点** 自由分配到任意技能上。技能等级 0-10，每提升 1 级消耗 1 点。

| 等级 | 水平 |
|---|---|
| 0 | 没碰过——纯靠属性硬扛 |
| 1-2 | 略懂。能辅助，别交给你负责 |
| 3-4 | 熟手。日常够用 |
| 5-6 | 专业。以此为生没人质疑 |
| 7-8 | 大师。在这个领域里会被提起名字 |
| 9-10 | 传奇 |

::: tip 分配心法

别摊薄。66 项技能，聚焦才有锋芒。

- **角色能力对应的技能投入 4-6 点**——你吃饭的家伙
- **察觉 (Perception) 和 闪避 (Evasion)** 是通用生存技能
- **至少 1-2 项社交/生活技能**——纯粹的战斗机器在夜之城寸步难行
- 反过来想：哪些技能你愿意是 0？那些你不碰的领域本身就在说你是谁

:::

## **六、你的钱意味着什么**

:::

RED 里，**钱不只是购物清单**——钱是重力。每个月都要付房租、吃饭、活着。没钱的时候，选择会替你做。

在你开始买东西之前，先把**要付的账**算清楚。与 DM 一起决定以下两项：

### 住处

| 住处 | 月租 | 什么样 |
|---|---|---|
| **街头/流浪** | 免费 | 桥洞、废墟、别人房檐下。不安全。冬天要命。 |
| **立方旅馆** Cube Hotel | 100eb | 胶囊床。公共卫浴。行李不丢就谢天谢地。 |
| **货柜屋** Cargo Container | 300eb | 改装集装箱。有门有锁。Combat Zone 边缘常见。 |
| **工作室公寓** Studio Apartment | 500eb | 独立卫浴和小厨房。中产区。 |
| **高级公寓** Upscale Apartment | 1000eb | 安全区。门禁。邻居不会半夜开枪。 |
| **企业公寓** Corporate Apartment | 1500eb+ | 企业提供或补贴。安全、干净、随时可能被收回。 |

### 伙食

| 伙食 | 月费 | 什么样 |
|---|---|---|

### 月度开支的意义

把住处和伙食的月费加起来——这个数字就是你**每个月必须挣到的最低线。**

低于这个线，你就得补：放弃治疗、欠租、借债、接不该接的活。

**钱不是等你买完装备才考虑的——钱是你买装备的时候就在算的。** 见了护甲表不要只看 SP，看看价格旁边你的月度开支。

## **七、你身上有多少还是你的**

:::

义体是 RED 最核心的博弈。**每装一件东西，你就离自己远一步。**

### 两个数字，两种代价

安装义体时，人性受到两层影响——两者不是一回事：

#### 1. 安装损耗 (Humanity Loss)——当前人性值下降

每件义体安装时，从你的**当前人性值**中扣减对应数值。这是**可以恢复的**——通过心理治疗。

#### 2. 上限缩减 (Permanent Reduction)——最大人性值降低

每件义体会**永久降低你的人性值上限**。标准义体 -2，大型义体（斯安威斯坦、克里多等）-4。这个缩减**不可通过治疗恢复**——除非你后来提升了 EMP 属性（EMP 提升时，上限重新计算：新 EMP × 10，再减去所有已安装义体的上限缩减）。

| 义体 | 安装损耗 | 上限缩减 | 效果 |
|---|---|---|---|
| **神经处理器** Neural Link | 7 | 2 | 安装额外插槽的先决条件 |
| **赛博义眼** Cybereye ×1 | 7 | 2 | 可插入视觉模块（红外、微光、瞄准、录像） |
| **赛博义耳** Cyberaudio Suite | 7 | 2 | 可插入听觉模块（增距、降噪、录音、无线电） |
| **义肢** Cyberarm | 7 | 2 | 可安装手臂插槽（隐藏武器、工具手、抓钩） |
| **义腿** Cyberleg | 7 | 2 | 可安装腿部插槽（跳跃增强、隐藏储物） |
| **皮肤编织** Skin Weave | 7 | 2 | 皮下植入。身体 SP 7，不叠加外穿护甲 |
| **移植物肌肉** Grafted Muscle | 7 | 2 | BODY 永久 +1 |
| **斯安威斯坦** Sandevistan | 14 | 4 | 激活后 REF 临时 +1（1min）。使用后强烈疲劳。 |
| **克里多** Kerenzikov | 14 | 4 | REF 永久 +2。永远处于轻微加速状态。 |

### 人性值阈值

| 当前人性值 | 状态 |
|---|---|
| 满（EMP × 10，减去上限缩减） | 完整的人性 |
| 低于上限 20+ | 开始对人冷漠。不是刻意——是感觉不到。 |
| < 30 | **边缘型人格障碍**。DM 可要求人格检定。 |
| ≤ 0 | **赛博精神病**。角色归 DM，变为 NPC。 |

### 恢复人性

- **心理治疗**：每周一次，恢复 `2d6` 当前人性值，花费 500eb/次。只能恢复到**被上限缩减之后的最大值**。
- **提升 EMP**：在游戏后期通过成长提升 EMP 属性——这会重新拉高你的人性值上限。但你装过的义体数量决定了你追不追得回来。

### 每次选择义体时

不要先看 HL 数字。先想：**你的角色为什么要在身上装东西？**

- 为了活命？（凯夫拉皮下——SP 7，但上限 -2）
- 为了比所有人快？（斯安威斯坦——14 安装损耗、上限 -4。发作时你是闪电，事后像被掏空。每次用都是赌博）
- 因为原来的身体不够用了？还是**已经不想要原来的身体了？**

两种理由都成立。后一种的代价是算不出来的——那是你的角色弧光。

::: info 经济与人性在此交汇

心理治疗费用 500eb/次。如果你每月的伙食是合成糊（50eb）+ 货柜屋（300eb），你的月度基础开支是 350eb。

一次治疗 = 一个半月的全部生活费。

而你在治的过程中可能还会装新东西。**这就是 RED：每一件义体都是一笔经济账，也是一笔人性账。两本账在同一页纸上。**

:::

> **★2077**：2077 使用义体容量槽位，不追踪 HL 和上限缩减。桌面 RED 保留完整人性规则。

## **九、你的钱从哪来**

:::

开局资金不只是数字——**来源**本身就是一个叙事选择。与 DM 商量你的角色状况：

::: tabs

@tab 街头出身

全靠自己。没靠山，没欠条。

- **资金**：DM 决定，通常 800-2550eb
- **意味着**：自由。没人控制你。但没人会在你弹尽粮绝时伸手。

@tab 公司走狗

你签了合同。企业给了起点——欠的要在游戏里还。

- **资金**：**2550eb 基础 + 企业附加包**。附加内容与 DM 协商：

| 企业附加 | 说明 |
|---|---|
| 企业公寓 | 住房已解决。月度住宿开支为 0。但公寓属于公司。 |
| 公司载具 | 通勤车。未必你名下。 |
| 额外义体预算 | 企业已为你安装基础件——通常为神经处理器或基础义眼。视作已安装，安装损耗和上限缩减照常扣除。 |
| 商务正装 | 一套。穿着它代表公司。 |
| 附属职责 | 定期汇报。任务优先。 |

- **意味着**：你不是为自己活。==企业的钱从来不免费。==

@tab 欠债起家

手里没钱——但有人愿意掏。

- **资金**：DM 根据债主决定额度。通常高于 2550eb。
- **意味着**：每一分钱都被人记着。债主可能是中间人、帮派头目、企业中层——DM 会在合适的时机触发这笔债。

:::

::: tip 不是选最富的

是选**最能推动角色故事的。** 街头是"从无到有"。公司是"什么时候撕合同"。欠债是"别人什么时候来讨"。三个都是好故事。钱在 RED 里从来不只是数字——**它是你每次选择背后的引擎声。**

:::
