## **技能与工具**

:::

[技能](../data/skill) 与 [工具](../data/tool) 代表着角色所擅长的、模糊的领域

因此相当广泛、通用，==适合即兴使用==，请尽情发掘其稀奇古怪的使用方式

既定的使用方式可以查看两者对应的页面

即兴时，可以理解为：`技能` 是日常大类的普遍运用，而 `工具` 是 `技能` 小类的精尖细分

-----

::: center

## **休整**

:::

玩家可以主动提出休整，分为 `短休` (1 小时) 与 `长休` (8 小时)

-----

::::: card

::: center

### **短休**

:::

#### **条件**
- 至少 1 点 `生命值`；
- 可进行轻度活动（阅读、交谈、进食或站岗）
- 持续 1 小时

-----

::: center

#### **恢复**
- 消耗 `生命值骰` 回血：恢复 <Badge type="danger" text="(职业生命骰 + 体质调整值) * 消耗个数" />，可陆续消耗
- 恢复资源：恢复对应特性、`法术` 等资源次数

:::

-----

::: right

#### **打断**
- 投掷 `先攻`
- 施展一道非 `戏法` 的 `法术`
- 受到任何伤害

:::

-----

::: center

#### **鉴定魔法物品**

`短休` 可用于保持接触、精神聚焦以 ==知晓单件魔法物品词条与使用方法（不含诅咒）==

-----

#### **同调**

`短休` 可用于保持接触、精神聚焦以 ==与单件魔法物品进行同调（最多同时存在 3 件 / 角色）==

:::

:::::

-----

::::: card

::: center

### **长休**

:::

#### **条件**
- 至少 1 点 `生命值`；
- 保证 6 小时睡眠（`昏迷`状态）
- 可进行 2 小时轻度活动（阅读、交谈、进食或站岗）
- 持续 8 小时
- `长休` 之间间隔 16 小时

-----

::: center

#### **恢复**
- 全部 `生命值`、`生命值上限`、`生命骰`
- `属性值`
- `力竭` 层数 -1
- 恢复资源：恢复对应特性、`法术` 等资源次数

:::

-----

::: right

#### **打断**
- 投掷 `先攻`
- 施展一道非 `戏法` 的 `法术`
- 受到任何伤害
- 1小时的行走或其他的体力劳动

:::

::: center

若打断前已满足 `短休`，则视为 `短休`

打断后可以增加额外 1 小时时间以继续长休

:::

:::::

-----

::: center

### **睡眠**

睡眠等同于 `昏迷` 状态

:::

::::: card

::: center

#### **唤醒**

:::

::: left

必然：
- 受到伤害
- 1 个动作的无声物理唤醒（摇晃、拍打等）
- 巨响（叫喊、雷鸣）

:::

-----

::: right

概率：
- 10ft 内耳语，且被动 `察觉` >= 20
- 安静环境（无风声、鸟鸣、蟋蟀等杂音）正常音量谈话，且被动 `察觉` >= 15

:::

:::::

-----

::::: card

::: center

#### **失眠**

24 hr 未 `长休`，`体质豁免` `DC` = 10，否则`力竭` + 1

每 +24 hr，`豁免 DC` +5

## **制作**

### **物品**

:::

::: center

制作需要角色 ==连续工作 8 小时== 算作 1 天工作量

其他 ==有条件== 的角色协助则令 8 小时 / 参与人数

:::

::: table

| **稀有度** | **熟练条件**   | **原料（金币）**    | **时间（天）** |
|:-------:|:----------:|:-------------:|:---------:|
| **非魔法** | `工具`       | 价格 / 2        | 价格 / 10   |
| **普通**  | `工具` +`奥秘` + (准备对应法术) | 原物品 + 50      | 5         |
| **非普通** | `工具` +`奥秘` + (准备对应法术) | 原物品 + 200     | 10        |
| **珍稀**  | `工具` +`奥秘` + (准备对应法术) | 原物品 + 2,000   | 50        |
| **极珍稀** | `工具` +`奥秘` + (准备对应法术) | 原物品 + 20,000  | 125       |
| **传奇**  | `工具` +`奥秘` + (准备对应法术) | 原物品 + 10,0000 | 250       |

:::

::: center

::: tip 魔法消耗品

除卷轴外，原料成本与制作时间均减半

:::

::: center

::: note 治疗药水

T1 的治疗药水使用：`草药工具` + 25 GP + 1 天 进行制作，即削减了制作天数

因此，可理解为：治疗药水皆只需 1 天制作

:::

### **法术卷轴**

:::

::: center

需要 `奥秘` 或 `书法工具` 的 `熟练` + 准备对应 `法术` + 对应 `法术` 材料成分（完成时消耗）

卷轴的施法者等级、`法术豁免DC` 与 `法术攻击加值` = 抄录完成时

| **`戏法`**   | 1         | 30         | 13         |    +5     |
| **1**    | 1         | 50         | 13         |    +5     |
| **2**    | 3         | 200        | 13         |    +5     |

| **4**    | 10        | 2,000      | 15         |    +7     |
| **5**    | 25        | 3,000      | 17         |    +9     |
| **6**    | 40        | 20,000     | 17         |    +9     |
| **7**    | 50        | 25,000     | 18         |    +10     |
| **8**    | 60        | 30,000     | 18         |    +10     |
| **9**    | 120       | 100,000     | 19         |    +11     |

::: info 使用卷轴

需要法表中有对应法术才可使用卷轴

### **武器**

:::

:::: card-grid

::: card

按 ==类型== 分为：`天生武器`、`临时武器`、`简易武器` 和 `军用武器`

:::

::: card

按 ==攻击方式== 分为：`徒手打击`、`近战武器攻击`、`投掷`、`远程武器攻击` 和 `法术攻击`

:::

::: card

`职业`等方式会提供 `受训` ，使用 `受训` 的武器进行攻击可以 ==添加 `熟练加值`==，反之则无加值（如，随手抄起非武器物品进行 `临时武器` 攻击）

:::

::: card

`职业`等方式会提供 `精通` ，使用 `精通` 的武器进行攻击可以 ==使用 `精通词条` 效果==

:::

::::

::: card

::: center

武器数据各有：分类、伤害骰、伤害类型、词条、精通词条、重量、价格

数据可参考：[武器](../data/item/common#武器)

:::

::: note 村规

不合身的武器可以使用下两种方式之一结算：
- 视为 `临时武器`，即 `攻击检定` 无 `熟练加值`，伤害骰 = <Badge type="danger" text="1d4" />， 投掷射程为（20/60）
- 视为对应 `武器`，但`攻击检定`劣势，伤害骰 = <Badge type="danger" text="武器伤害骰" />，但按照体型增减骰子规格，如：大一型，d6 -> d8

### **照明**

:::

::::: card

::: center

#### **视觉**

:::

:::: card-grid

::: card title="黑暗视觉"

- `微光` = `明亮`
- `黑暗` = `微光`
- 无法分辨颜色
- 无法看穿魔法造成的 `黑暗`

:::

::: card title="真实视觉"

- 看穿 `黑暗`，包括普通与魔法
- 可见 `隐形`
- 幻象变透明，自动 `豁免`
- 看出变化真实形态
- 可见 `以太位面`

:::

::: card title="盲视"

- 不在 `全身掩护` 的目标

- `目盲`、`黑暗` 不影响

- 可见 `隐形`

:::

::: card title="震颤感知"

- 生物/移动物件的方位
- 同一表面/液体接触（无法侦测空中）

:::

船只：

- 在无风中
    - 无法通过船帆移动、依靠船桨行驶
- 强逆风中
    - 仅能以半速行驶

---

- 在顺流时
    - 获得等同于水流流速的提升
    - 通常是 3 里 / 时
- 强逆流时
    - 无法通过船桨逆行

1 mile ≈ 1.6 km

追逐中，疾走动作可用次数为：<Badge type="warning" text="3 + 体质调整值(至少为 1)" />

使用更多的疾走动作，则需要在`回合`结束时进行 `体质``豁免``DC` = 10，否则获得 1 级`力竭`

`力竭`到达 5 级时，退出追逐

该途径获得的`力竭`可在`短休`或`长休`恢复
