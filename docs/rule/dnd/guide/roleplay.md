## **技能与工具**

:::

[技能](../data/skill) 与 [工具](../data/tool) 代表着角色所擅长的、模糊的领域

因此相当广泛、通用，==适合即兴使用==，请尽情发掘其稀奇古怪的使用方式

既定的使用方式可以查看两者对应的页面

即兴时，可以理解为：`技能` 是日常大类的普遍运用，而 `工具` 是 `技能` 小类的精尖细分

往往 `技能` 是 ==单独使用== 的，其 `熟练` 通常也是一种 ==参与检定的前提门槛==

而 `工具` 可与 `技能` ==搭配使用==，当有可以对应的 `工具` ==辅助 `技能`== 时，不妨大胆提出

## **休整**

:::

玩家可以主动提出休整，分为 `短休` (1 小时) 与 `长休` (8 小时)


-----

<Card>

::: center

### **短休**

:::

#### **条件**
- 至少 1 点 `生命值`；
- 可进行轻度活动（阅读、交谈、进食或站岗）
- 持续 1 小时

-----

::: center

#### **恢复**
- 消耗 `生命值骰` 回血：恢复 <Badge type="danger" text="(职业生命骰 + 体质调整值) * 消耗个数" />，可陆续消耗
- 恢复资源：恢复对应特性、`法术` 等资源次数

:::

-----

::: right

#### **打断**
- 投掷 `先攻`
- 施展一道非 `戏法` 的 `法术`
- 受到任何伤害

:::

-----

::: center

#### **鉴定魔法物品**

`短休` 可用于保持接触、精神聚焦以 ==知晓单件魔法物品词条与使用方法（不含诅咒）==

-----

#### **同调**

`短休` 可用于保持接触、精神聚焦以 ==与单件魔法物品进行同调（最多同时存在 3 件 / 角色）==

:::

</Card>

-----

<Card>

::: center

### **长休**

:::

#### **条件**
- 至少 1 点 `生命值`；
- 保证 6 小时睡眠（昏迷状态）
- 可进行 2 小时轻度活动（阅读、交谈、进食或站岗）
- 持续 8 小时
- `长休` 之间间隔 16 小时

-----

::: center

#### **恢复**
- 全部 `生命值`、`生命值上限`、`生命骰`
- `属性值`
- `力竭` 层数 -1
- 恢复资源：恢复对应特性、`法术` 等资源次数

:::

-----

::: right

#### **打断**
- 投掷 `先攻`
- 施展一道非 `戏法` 的 `法术`
- 受到任何伤害
- 1小时的行走或其他的体力劳动

:::

::: center

若打断前已满足 `短休`，则视为 `短休`

打断后可以增加额外 1 小时时间以继续长休

:::

::: warning 村规改动

村规对 `长休` 做了比较大的改动，以增添资源管理难度，请参照 [扎营长休](../shinkan42/camp.md)

:::

</Card>

-----

::: center

### **睡眠**

睡眠等同于 `昏迷` 状态

:::

<Card>

::: center

#### **唤醒**

:::

::: left

必然：
- 受到伤害
- 1 个动作的无声物理唤醒（摇晃、拍打等）
- 巨响（叫喊、雷鸣）

:::

-----

::: right

概率：
- 10ft 内耳语，且被动 `察觉` >= 20
- 安静环境（无风声、鸟鸣、蟋蟀等杂音）正常音量谈话，且被动 `察觉` >= 15

:::

</Card>

-----

<Card>

::: center

#### **着甲**

`扎营` 村规中的 “护甲穿脱” 简易行动可以减去 1 层护甲，如：重甲 -> 中甲

:::

::: flex center

| **护甲层级** | **生命值**    | **生命骰**    | **力竭** | **资源** |
|:--------:|:----------:|:----------:|:------:|:------:|
| **轻甲**   | /          | /          | /      | /      |
| **中甲**   | 1/2        | 1/2        | -1     | 1/2    |
| **重甲**   | 1/2 且不享受医疗 | 1/2 且不享受医疗 | -2     | 1/2    |


:::


::: warning 村规注意

原规则为：中甲/重甲 `长休` 只回复 1/4 `生命骰` 且不恢复 `力竭`

`力竭` 部分从 “不恢复” 变为 “-1”，等效但是与 `扎营` 村规联动后，后果可更重

:::


</Card>

-----

<Card>

::: center

#### **失眠**

24 hr 未 `长休`，体质豁免 DC = 10，否则`力竭` + 1

每 +24 hr，豁免 DC +5

:::

</Card>

## **制作**

### **物品**

:::

::: center

制作需要角色 ==连续工作 8 小时== 算作 1 天工作量

其他 ==有条件== 的角色协助则令 8 小时 / 参与人数

:::

::: flex center

| **稀有度** | **熟练条件**   | **原料（金币）**    | **时间（天）** |
|:-------:|:----------:|:-------------:|:---------:|
| **非魔法** | `工具`       | 价格 / 2        | 价格 / 10   |
| **普通**  | `工具` +`奥秘` + (准备对应法术) | 原物品 + 50      | 5         |
| **非普通** | `工具` +`奥秘` + (准备对应法术) | 原物品 + 200     | 10        |
| **珍稀**  | `工具` +`奥秘` + (准备对应法术) | 原物品 + 2,000   | 50        |
| **极珍稀** | `工具` +`奥秘` + (准备对应法术) | 原物品 + 20,000  | 125       |
| **传奇**  | `工具` +`奥秘` + (准备对应法术) | 原物品 + 10,0000 | 250       |

:::

::: center

::: tip 魔法消耗品

除卷轴外，原料成本与制作时间均减半

:::

::: center

::: note 治疗药水

T1 的治疗药水使用：`草药工具` + 25 GP + 1 天 进行制作，即削减了制作天数

因此，可理解为：治疗药水皆只需 1 天制作

:::

### **武器**

:::

:::: card-grid

::: card

按 ==类型== 分为：`天生武器`、`临时武器`、`简易武器` 和 `军用武器`

:::

::: card

按 ==攻击方式== 分为：`徒手打击`、`近战武器攻击`、`投掷`、`射击` 和 `法术攻击`

:::

::: card

职业等方式会提供 `受训` ，使用 `受训` 的武器进行攻击可以 ==添加 `熟练加值`==，反之则无加值（如，随手抄起非武器物品进行 `临时武器` 攻击）

:::

::: card

职业等方式会提供 `精通` ，使用 `精通` 的武器进行攻击可以 ==使用 `精通词条` 效果==

:::

::::

::: card

::: center

武器数据各有：分类、伤害骰、伤害类型、词条、精通词条、重量、价格

数据可参考：[点击跳转](../data/equipment/weapon.md)

:::

::: note 村规

不合身的武器可以使用下两种方式之一结算：
- 视为 `临时武器`，即 `攻击检定` 无 `熟练加值`，伤害骰 = <Badge type="danger" text="1d4 + 力量调整值" />， 投掷射程为（20/60）
- 视为对应 `武器`，但`攻击检定`劣势，伤害骰 = <Badge type="danger" text="武器伤害骰" />，但按照体型增减骰子规格，如：大一型，d6 -> d8

### **照明**

:::

<Card>

::: center

#### **视觉**

:::

:::: card-grid

::: card title="黑暗视觉"

- `微光` = `明亮`
- `黑暗` = `微光`
- 无法分辨颜色
- 无法看穿魔法造成的 `黑暗`

:::

::: card title="真实视觉"

- 看穿 `黑暗`，包括普通与魔法
- 可见 `隐形`
- 幻象变透明，自动 `豁免`
- 看出变化真实形态
- 可见 `以太位面`

:::

::: card title="盲视"

- 不在 `全身掩护` 的目标

- `目盲`、`黑暗` 不影响

- 可见 `隐形`

:::

::: card title="震颤感知"

- 生物/移动物件的方位
- 同一表面/液体接触（无法侦测空中）

:::

::::

</Card>

:::

普通的黑暗视觉无法看穿
